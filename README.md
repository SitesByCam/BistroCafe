🍽️ Bistro Café Website (Mock Template)

A modern, responsive bistro café website template created to showcase clean design, user experience, and front-end development skills.

🔗 Live Demo:
https://sitesbycam.github.io/BistroCafe/

📌 Overview

This project is a mock website designed as a reusable template for small café or restaurant businesses. It focuses on a clean, professional layout while maintaining a warm and inviting aesthetic that reflects a real-world bistro environment.

The site demonstrates how a business can effectively present its menu, atmosphere, and key information through a structured and user-friendly design.

✨ Features
Fully responsive design (mobile, tablet, desktop)
Clean and modern UI with a warm café aesthetic
Smooth navigation across multiple pages
Menu layout with structured sections
Image gallery with lightbox functionality
Contact page with directions and map integration
Reusable layout suitable for real client projects
🛠️ Built With
HTML5
CSS3 (Flexbox, Grid, responsive design)
JavaScript (UI interactions & lightbox)
Google Maps API (directions feature)

This project was created as part of my portfolio under SitesByCam, with the goal of demonstrating:
